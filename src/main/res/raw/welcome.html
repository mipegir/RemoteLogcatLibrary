Bienvenido al Log de #APP_NAME#!

Cómo utilizar este Log:

Para ver el log que ha generado tu dispositivo, sólo tienes que concatenar a tu actual URL el path /log, con ello ya tendrás acceso para ver lo que ocurre en tu dispositivo.

Pero quizá ver toda esta información al mismo tiempo puede ser algo engorroso, así que puedes ir un poco más allá filtrando por líneas:

  • Puedes filtrar obteniendo sólo las líneas que empiezan con cierta cadena con filterStart.
  • Puedes filtrar obteniendo sólo las líneas que contengan una cadena concreta con filterContains.

Ejemplos de filtrado:

Opciones de filtrado Ejemplo
filterStart http://IP_DEVICE:8080/log?filterStart=D/OkHttp
filterContains http://IP_DEVICE:8080/log?filterContains=appStatus
